Linux - Comandos do Shell

Comando Descrição
cd Selecionar diretório.
ls "dir" Lista os arquivos do diretório "dir"
mkdir "dir" Cria o diretório "dir".
mv "file" "dir" Move o arquivo "file" para o diretório "dir"
touch "file" Cria o arquivo "file".
chmod XYZ "file" Altera as permissões de um arquivo "file" para números X,Y e Z .
chown $USER "file" Altera o proprietário de um arquivo "file".
chown -R $USER:$GROUP . Altera o proprietário de todos os arquivos do diretório atual.
